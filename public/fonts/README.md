# Font placeholder — rischio licenza

Questi file Helvetica (.ttf) vengono da
`Desktop/03_MAGISTRALE_ADVANCED_DESIGN/design_sostenibilita/helvetica-255-cufonfonts/`.
Nessun file di licenza/EULA è presente accanto ad essi, e "cufonfonts" è un sito
noto per distribuire font senza chiara licenza d'uso commerciale/web.

**Non pubblicare il sito in produzione con questi file.** Sono qui solo per
sviluppare/vedere il sistema tipografico in locale. Prima del lancio pubblico
serve una delle due cose:

1. Licenziare Helvetica Neue/Now per il web (Monotype, Linotype, Adobe Fonts).
2. Sostituire con un font regolarmente licenziato dal disegno equivalente
   (es. Helvetica Now Text via Adobe Fonts, o alternativa open-source molto
   simile se il budget non lo consente).
